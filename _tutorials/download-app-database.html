In this short tutorial you will learn how to make backups of your app data and import it on another device.

To import or export your history, subscriptions and playlists, open the settings and go to the Content category. Scroll down and you will see two entries:

  • Export database
  • Import database

Export your app data

To export your app data click on Export database. You will now see a dialog where you can select the directory in which NewPipe should store the export file. After selecting a directory and confirming it, NewPipe creates a ZIP-file for you.

This file is name like NewPipeData-YearMonthDay_Time.zip. This naming allows you to have several backups in one directory.
