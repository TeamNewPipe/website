You can either follow the instructions inside NewPipe or these ones. This tutorial is included in NewPipe too. It shows up, after opening the import / export dialog and selecting the action you want to perform.

To export your subscriptions, swap to the subscriptions tab, tap on Import/Export. Then just click on Export to File, select a directory, pick a filename and confirm.

To import you subscriptions you first need to have a file which you can import. There are three different ways you can get this file:
